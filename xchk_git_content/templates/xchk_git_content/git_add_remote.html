Schriftelijk materiaal

Als je tot dit punt in de leerstof bent geraakt, heb je al een lokale repository en een remote op het xchk systeem. Je kan meerdere remotes hebben. Typisch staan je remotes op een hosting dienst, zoals Bitbucket.

Je verkrijgt een repository via git init of git clone. In het eerste geval start je zonder remote, in het tweede geval heb je een remote. Als je nog geen remote hebt en er een wil toevoegen, volg dan het voorbeeld in het filmpje. Het filmpje maakt gebruik van Bitbucket, maar het proces is gelijkaardig voor andere diensten.
